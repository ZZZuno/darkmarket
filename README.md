# Spring-Project-DarkMarket
스프링 + JSP 파싱_운동식단쇼핑몰


## 프로젝트 소개
운동인을 위한 여러 식단을 판매하는 쇼핑몰을 만들었습니다.
<br>

## 개발 기간
* 23.11.17 ~ 23.12.27

### 개발 환경
- `Java 8`
- `JDK 1.8.0`
- **IDE** : STS 4.7
- **Framework** : Spring(MVC)
- **Database** : Oracle DB(11gXE)
- **ORM** : Mybatis

## 주요 기능
#### 로그인
- DB값 검증
- ID, PW 찾기
- 로그인 시 세션(Session) 생성
